On first launch, Nori performs a one-time setup: creates the config directory, probes your default shell, installs Nerd Font (Windows), and initializes the command history file. Subsequent launches are sub-50ms.

<Code label="Launch" lang="bash">{`nori

# Directly from source build
./target/release/nori

# Windows (from build directory)
.\\target\\release\\nori.exe`}</Code>
