### Input Modes

Typing <InlineCode>/git</InlineCode> alone enters **Git mode** — all subsequent commands are automatically prefixed with <InlineCode>/git</InlineCode>. Same for <InlineCode>/docker</InlineCode>. The prompt marker changes to indicate the active mode. Type <InlineCode>exit</InlineCode> to return to normal mode.
